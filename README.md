# Manual Site Institucional Grupo +B

Manual HTML interativo para orientar designers, UI/UX, programadores, social media e IAs na criação do site institucional do Grupo +B.

## Arquivo principal

Abra o arquivo `index.html`.

## Contexto

O protagonista do projeto é o Grupo +B, também chamado Grupo Mais Barato. O +B Supermercados é a principal marca atual do portfólio, mas não deve ser tratado como o grupo inteiro.

## Direção visual

- Fundo preto ou grafite
- Amarelo #FFC400 como cor de destaque
- Vermelho #F20D18 com uso controlado
- Tipografia DM Sans
- Visual premium acessível
- Layout limpo, didático e institucional
